nubotv-themeemetv - Streaming UI Theme

A modern and responsive **streaming platform UI theme** inspired by popular OTT platforms like Netflix and Prime Video.

---

## 🚀 Features

* 🎥 Clean and modern streaming UI design
* 📱 Fully responsive (mobile, tablet, desktop)
* 🌙 Dark theme optimized for media viewing
* ⚡ Fast and lightweight design
* 🎨 Customizable components
* 📺 Movie/series style layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3 / Tailwind CSS
* JavaScript

---

## 📂 Project Structure

```
nubotv/
│── assets/        # Images, icons, media
│── css/           # Stylesheets
│── js/            # JavaScript files
│── index.html     # Main entry file
│── components/    # UI components (cards, navbar, etc.)
```

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/DevAizenx/nubotv.git
```

2. Open the project folder:

```
cd nubotv
```

3. Run in browser:

* Open `index.html` directly
* Or use Live Server (VS Code recommended)
